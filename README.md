# Claude Code plugins

Public plugins by Mikko Parkkola. Add the marketplace once, then install any of them:

```bash
claude plugin marketplace add MikkoParkkola/claude-marketplace
claude plugin install trvl          # or: glyphdown, anti-ai-tell
```

| Plugin | What it does |
|--------|--------------|
| `trvl` | Travel search and planning -- flights, hotels, trains, ferries, price alerts. No API keys. |
| `glyphdown` | Token-cost reduction for Claude Code -- lossless output compression, stacks on prompt caching. |
| `anti-ai-tell` | Writing discipline that keeps prose from reading as AI-generated. |

Each plugin lives in its own repo; this marketplace just points at them. More tools: https://github.com/MikkoParkkola
